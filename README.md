# Czech Republic Food Prices and Payroll Analysis

A data analysis project created as part of a **Data Analysis course**, combining **SQL**, **Power BI**, and an additional self-directed **Python analysis** focused on long-term trends in food prices, GDP, and average wages in the Czech Republic between **2006 and 2018**.

## Project Background

This project was originally created during a **Data Analysis course**.

The main goal of the course assignment was to:
- analyze the dataset using **SQL**
- answer a set of analytical questions
- create a **Power BI dashboard** based on the processed data

The analytical questions and written answers are included in:

**`accompanying_document.pdf`**

After finishing the original assignment, I decided to extend the project on my own by adding an additional **Python analysis**.

## Project Goal

The project had two main objectives:

### 1. Course assignment
- prepare and analyze the data in SQL
- answer analytical questions based on the dataset
- build a Power BI dashboard to present the results

### 2. Personal extension
- perform additional analysis in Python
- visually explore long-term development of selected food prices
- compare food price trends with **GDP** and **average wages** in the Czech Republic

## Dataset

The dataset covers the period **from 2006 to 2018**.

It includes data related to:
- food prices in the Czech Republic
- average wages in the Czech Republic
- GDP development in the Czech Republic

## Repository Contents

- `primary_table.sql` – creation of the main analytical table
- `secondary_table.sql` – creation of the secondary table
- `question_one.sql` to `question_five.sql` – SQL queries answering the analytical questions
- `SQL project.csv` – exported dataset used in the project
- `Data_HDP_pandas.csv` – dataset used for the Python analysis
- `Pandas.ipynb` – additional Python analysis
- `Data_analysis_prices.pbix` – Power BI dashboard
- `Data_analysis_prices.pdf` – PDF export of the Power BI dashboard
- `accompanying_document.pdf` – analytical questions and written answers

## SQL and Power BI Part

The main part of the project was completed within the data analysis course.

The objective was to:
- process the data in SQL
- answer analytical questions
- prepare outputs for dashboarding in Power BI

The final dashboard focuses on the development of wages and food prices in the Czech Republic and allows users to explore the data from several perspectives.

## Dashboard Features

The Power BI dashboard includes:
- **dynamic titles** based on user selections
- yearly wage analysis across industries
- food price comparison in selected years
- trend analysis of selected food categories over time
- purchasing power perspective based on average wages
- comparison of **GDP**, **food prices**, and **average wages** over time

The dashboard was later improved with more dynamic and interactive elements, especially dynamic page titles responding to slicer selections.

## Python Extension

After completing the original course assignment, I extended the project with my own **Python analysis**.

The main goal of this extension was to **graphically visualize selected long-term trends** rather than recreate the whole SQL/Power BI workflow in Python.

The Python notebook focuses mainly on:
- trends in selected food prices
- GDP development in the Czech Republic
- average wage development in the Czech Republic

This part of the project helped me practice Python on a real dataset and view the same topic from a different analytical perspective.
